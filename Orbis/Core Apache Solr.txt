Creacion de Sitio Apache Solar
==============================
	ingresar a la bdr32
	cd /data/solr/
	cp ambiente ambiente nuevo ---> copiar el ambiente a  uno nuevo
	chown -R tomcat7:tomcat7 ----> cambiar los permisos para usuario y grupo
	rsync -az ambiente nuevo_ambiente  ----> se sincroniza la carpeta
	Editar los siguientes archivos:
		vim /path/core.propieter
		vim /path/conf/schema.xml

	Apuntar el Sitio
	----------------
	cd /usr/share/tomcat/webapps/
	cp -R ambiente nuevo_ambiente
	vim ambiente/web-inf/web.xml

	Modificacion de Linea
		<env-entry-value>/data/solr/ambiente </env-entry-value>

	Reiniciar el tomcat
		/etc/init.d/tomcat restart

	Ingresar a Solr en el navegador
		

	git clone git@git.ec.pe:aptitus3-nodejs.git -o origen dev.node1a.aptitus.com
