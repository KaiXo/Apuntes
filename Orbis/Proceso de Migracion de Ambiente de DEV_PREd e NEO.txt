Proceso de Migracion de Ambiente de DEV/PREd e NEOAUTO
++++++++++++++++++++++++++++++++++++++++++++++++++++++

Consideraciones: Instalacion libreria de ImageMagick
++++++++++++++++++++++++++++++++++++++++++++++++++++
1.- Version de Apache y PHP debe estar homologado con la version de PRE/DEV
2.- Se debe de trabajar con la misma version de PRODUCCION que en este caso de la WEB01 es ImageMagick 6.7.6-6
Nota: esta version no esta disponible vamos a trabajar la version ImageMagick 6.7.6-9 de la pagina oficial -> http://sourceforge.net/projects/imagemagick/files/old-sources/6.x/6.7/
3.- Descargar el archivo: en el Server
 curl http://downloads.sourceforge.net/project/imagemagick/old-sources/6.x/6.7/ImageMagick-6.7.6-9.tar.gz
4.- Descomprimir el archivo
 tar xvfz ImageMagick.XXXX.XXX.tar.gz
5.- Proceso de Instalacion
 Ingresas a la carpeta
  cd ImageMagick-6.7.6-9
  ./configure
  make
  make install
6.- Instalamos
 yum install ImageMagic ImageMagic-devel
7.- Instalacion de Extencion para PHP
 pecl install imagick
 - Agregamos la linea siguinte en el archivo de php.ini en ->/etc/php.ini
 extension=imagick.so
8.- Reiniciar Apache 
 service httpd restart
